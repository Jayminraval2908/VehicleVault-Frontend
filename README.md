
# рҹҡ— Vehicle Vault

Vehicle Vault is a full-stack MERN (MongoDB, Express, React, Node.js) web application designed to simplify the process of buying and selling vehicles. It provides a secure and user-friendly platform where users can list vehicles, explore available listings, send inquiries, schedule test drives, and negotiate deals efficiently.


## рҹ“ҢProject Overview
Vehicle Vault connects buyers and sellers in a seamless digital marketplace. Sellers can upload vehicle details with images, while buyers can browse listings, make offers, and communicate directly through the platform.

The system ensures transparency, smooth deal management, and an enhanced user experience.

## рҹҡҖGetting Started

#### Installation


## Clone the Frontend repository
git clone https://github.com/Jayminraval2908/VehicleVault-Frontend.git

## Clone the Backend repository
git clone https://github.com/Jayminraval2908/VehicleVault-Backend.git

## Install backend dependencies
cd vehicle-vault-backend

npm install

## Install frontend dependencies
cd ../vehicle-vault-frontend

npm install


## Run the Project


## Start backend
cd vehicle-vault-backend

npm run dev

## Start frontend
cd vehicle-vault-frontend

npm run dev
