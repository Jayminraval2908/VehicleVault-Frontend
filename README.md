
# 🚗 Vehicle Vault

Vehicle Vault is a full-stack MERN (MongoDB, Express, React, Node.js) web application designed to simplify the process of buying and selling vehicles. It provides a secure and user-friendly platform where users can list vehicles, explore available listings, send inquiries, schedule test drives, and negotiate deals efficiently.


## 📌Project Overview
Vehicle Vault connects buyers and sellers in a seamless digital marketplace. Sellers can upload vehicle details with images, while buyers can browse listings, make offers, and communicate directly through the platform.

The system ensures transparency, smooth deal management, and an enhanced user experience.
## ✨Features
### 👤User Authentication

▪️Secure login/signup (JWT-based authentication)

▪️Google OAuth integration

▪️Role-based access (Buyer & Seller)

### 🚘Vehicle Management

▪️Add, update, and delete vehicle listings

▪️Upload multiple vehicle images

▪️Detailed vehicle information (price, model, condition, etc.)

### 💰Offer System

▪️Buyers can send offers to sellers

▪️Sellers can accept or reject offers

▪️Real-time offer status tracking

### 💬Inquiry System

▪️Buyers can send inquiries about any vehicle

▪️Sellers receive inquiries via dashboard/email

▪️Helps users ask questions before making an offer

### ⏳Test Drive Booking System

▪️Buyers can request a test drive for listed vehicles

▪️Sellers can approve or reject test drive requests

▪️Schedule date & time for test drive

▪️Track test drive status (Pending / Approved / Rejected)

### 📞Contact System

▪️Users can send queries directly to platform admin

▪️Email notification integration

### 📊Dashboard System

▪️Personalized dashboard for buyers and sellers

▪️Manage vehicles, offers, inquiries, and test drives in one place

## 🚀Getting Started

#### Prerequisites

Node.js |
MongoDB |
npm or yarn

#### Installation


## Clone the Frontend repository
git clone https://github.com/your-username/vehicle-vault.git

## Clone the Backend repository
git clone https://github.com/your-username/vehicle-vault.git

## Install backend dependencies
cd vehicle-vault-backend

npm install

## Install frontend dependencies
cd ../vehicle-vault-frontend

npm install


## Run the Project


## Start backend
cd vehicle-vault-backend

npm run dev

## Start frontend
cd vehicle-vault-frontend

npm run dev
